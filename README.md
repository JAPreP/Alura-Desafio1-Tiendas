#  Alura Store – Análisis de Datos de Ventas

##  Propósito del Proyecto

El objetivo de este proyecto es realizar un **análisis exploratorio de datos (EDA)** sobre las ventas de cuatro tiendas de la cadena **Alura Store**, con el fin de **identificar cuál de ellas debería ser vendida** para que el Sr. Juan pueda iniciar un nuevo emprendimiento.

La recomendación final se basa en el análisis de múltiples indicadores clave, tales como ingresos, categorías de productos, calificaciones de clientes, costos de envío, evolución temporal de las ventas y distribución geográfica.

Este proyecto forma parte de un **desafío académico de Alura LATAM** enfocado en el desarrollo de habilidades en **Ciencia de Datos con Python**.

##  Análisis Realizados

Durante el desarrollo del proyecto se llevaron a cabo los siguientes análisis:

- **Análisis de facturación total** por tienda.
- **Cantidad de productos vendidos por categoría**.
- **Productos más y menos vendidos** en cada tienda.
- **Calificación promedio de los clientes**.
- **Costo promedio de envío** por tienda.
- **Evolución mensual de las ventas**.
- **Análisis geográfico** usando latitud y longitud.
- **Visualizaciones comparativas** para apoyar la toma de decisiones.

---

## Ejemplos de Gráficos e Insights

Algunos de los gráficos generados incluyen:

-  **Gráficos de barras** para comparar ingresos, categorías y productos más vendidos.
-  **Gráficos circulares (pie charts)** para visualizar la distribución de categorías.
-  **Gráficos de líneas** para analizar la evolución temporal de las ventas.
-  **Mapas geográficos** para observar la concentración espacial de las ventas.

###  Insight clave:
Aunque las cuatro tiendas presentan indicadores similares, el análisis integral permitió identificar una tienda con **menor aporte estratégico**, lo que facilita la recomendación de venta sin afectar significativamente el desempeño global del negocio.
